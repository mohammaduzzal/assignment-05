Important FAQ

What is the Document Object Model (DOM)?

DOM is a programming interface for html document.it is like a bridge that connects your HTML to javascript,so that you can interact with and change the content,structure and style of your page using code.

How do you select an element from the DOM?

Ways to Select Elements from the DOM:
1. document.getElementById()
This method selects an element by its ID.
2. document.querySelector()
This method selects the first element that matches a CSS selector (like a tag name, class, or ID)
3. document.querySelectorAll()
This method selects all elements that match a given CSS selector. It returns a NodeList, which is like an array of elements.
4. document.getElementsByClassName()
This method selects all elements with a specific class name and returns an HTMLCollection (similar to an array).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a way to handle events (like clicks) using a single event listener on a parent element instead of adding separate listeners to each child element.
usefulness:
1.Fewer Event Listeners:
You only need one listener on a parent element instead of adding one to each child element. This saves memory and makes your code run faster.
2.Handles New Elements Automatically:
If you add new child elements later, the single listener on the parent will still work for them. You donâ€™t have to set up new listeners each time.
3.Easier to Maintain:
Managing one event listener is simpler than managing many. It makes your code cleaner and easier to read
4.Flexibility:
You can easily handle different actions based on which child element was clicked. This means you can have one place in your code to handle multiple elements.

How do you manipulate an element's attributes and styles using the DOM?

1. To use getAttribute(), setAttribute(), and removeAttribute() for attributes.
2. Use the style property and classList for styles.
3. You can access attributes directly for common properties (like id and className).
